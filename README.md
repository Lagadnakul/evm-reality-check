# 📖 About

EVM Reality Check is an interactive educational website built to explain how an Electronic Voting Machine (EVM) conceptually works through visual simulations and engaging animations.

Instead of lengthy articles or debates, the project presents information in an interactive format that encourages users to explore system architecture, voting workflow, and common misconceptions.

The website is intended purely for educational purposes.

Learn about multiple conceptual layers including:

- Physical Security
- Operational Procedures

## 💻 Hacker Mode

A fun interactive parody inspired by the popular internet claim that:

> "3 syntax errors are enough to hack an EVM."

The simulation demonstrates why real-world systems involve significantly more than fixing code syntax.

Users can download a fun completion certificate after finishing Hacker Mode.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/evm-reality-check.git
```

Go inside project

```bash
cd evm-reality-check
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build production version

```bash
npm run build
```

This website is an educational and conceptual project.
